# Vue 3.2 + TypeScript + Pinia + Element-Plus 管理系统

### 前言 📖

本系统的样式及部分交互逻辑是模仿 https://github.com/HalseySpicy/Geeker-Admin 这个优秀的vue3后台管理系统，代码基本上是自己手撕的，只写了一个总体框架模板，目的只是为了学习vue3的组合式API写法以及typescript。

### 安装使用步骤

```text
# 安装依赖
npm install

# 启动项目
npm run dev

# 打包项目
npm run build:dev
npm run build:pro
```
